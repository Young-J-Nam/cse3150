HOW TO RUN:

TO TEST FRACTIONS:
g++ -std=c++17 testFractions.cpp -o fracTest
./fracTest 

TO TEST TIME:
g++ -std=c++17 testTime.cpp -o timeTest
./timeTest 



TO RUN WITH INPUTS IN COMMAND LINE:

TO RUN FRACTIONS:
g++ -std=c++17 runFractions.cpp -o runFrac
./runFrac

TO RUN TIME:
g++ -std=c++17 runTime.cpp -o runTime
./runTime
